# <span style="color:olive">Python vol.2 - Data types</span> 
---

### <span style="color:tan">Introduction</span> 

1. Data types are a <b><span style="color:SeaGreen">way to classify data items</span></b>. 

2. Data types represent the kind of value, which <b><span style="color:SeaGreen">determines what operations can be performed on that data</span></b>.

---

### <span style="color:tan">Python data types</span> 

![localImage](/assets/images/python/data_types/2.png)

1. In Python programming, a data type is <b><span style="color:SeaGreen">the classification of data that defines the possible operations that can be done on the object</span></b>.

2. Python has <b><span style="color:SeaGreen">various built-in types in the interpreter</span></b>: numerics, sequences, mappings, classes, instances and exceptions.

3. <b><span style="color:SeaGreen">For the each of data types Python has a dedicated keyword</span></b> eg., int for Integer, str for String, float for Floating Point, dic for Dictionaries, boold for Booleans and so on.
